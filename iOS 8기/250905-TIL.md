### 1. 스탠다드반 실습 과제 진행
[tableView(\_:didSelectRowAt:)](https://developer.apple.com/documentation/uikit/uitableviewdelegate/tableview(_:didselectrowat:))

어제 테이블뷰 수업 들으면서 어느정도 이해는 되었지만 혼자 과제를 해보려니까 하나도 기억이 안 나고 막막하기만 했다. 이 블로그를 참고하면서 같이 공부하며 실습을 해보았다. 블로그대로 공부하니 실습 때 배운 내용이 다시 생각났다.

[TableViewCell에 있는 버튼을 delegate를 이용하여 ViewController에서 설정하기](https://jkh0013011.tistory.com/entry/TableViewCell안에-있는-버튼을-delegate를-이용하여-viewController에서-설정하기)

#### 과제 진행과정
1. `Protocol` 파일을 하나 생성해서 그곳에서 프로토콜을 선언해주었다.
2. `FirstViewController`에서 프로토콜 채택하기. 코드 정리를 위해서 `extension`에 넣어주면 깔끔해진다.
```swift
extension FirstViewController: FirstTableViewCellDelegate {
		func buttonClicked(data: String) {
			datas.append(data)
			tableView.reloadData()
		}
}
```
3. `FirstTableViewCell`에 delegate 선언 및 버튼 클릭 함수 생성하기
```swift
weak var delegate: FirstTableViewCellDelegate?

@objc func buttonTapped() {
	let data = dataLabel.text ?? ""
	delegate?.buttonClicked(data: data)
}
```
` ?? ""`로 옵셔널 바인딩!

Git이 버전 관리하지 않아야 할 파일이나 폴더를 지정해주는 역할

### 5. 스택뷰 반복문 중첩하기
오늘 스택뷰의 반복문 코드를 구현하면서 2차원 배열을 구현해보았다.
```swift
pritvate let nums = [
	["7", "8", "9", "+"]
    ["4", "5", "6", "-"]
    ["1", "2", "3", "*"]
    ["AC", "0", "=" "/"]
]
```

계산기 버튼 하나하나 만들기에 너무 코드가 길어질 것 같아 수직 스택뷰를 반복시키고 수평 스택뷰를 반복시키기로 결정하였다. 그러면서 2차원 배열 반복문을 배우게 되었다.

```swift
for row in nums {
	let hStackView = UIStackView()
    
    // HorizontalStackView 레이아웃
}
```
행 반복문! `HorizontalStackView`를 vertical로 반복한다.

```swift
for title in row {	
	let button = UIButton()
    
    // button 레이아웃
}
hStackView.addArrangedSubview(button)
mainButtons.append(button)
```

행 안의 배열을 반복해준다. 반복문을 돌면서 button을 `hStackView`에 추가한다.
