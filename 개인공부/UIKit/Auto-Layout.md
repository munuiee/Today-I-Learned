# 🤔 Auto Layout
제약 조건(Constraints)에 따라 뷰 계층 구조에 있는 모든 뷰의 크기와 위치를 동적으로 지정하는 것 <br>
위치나 크기를 다른 객체(Safe Area)를 이용해 상대적으로 제약을 준다.
